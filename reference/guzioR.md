# Internal package methods

These include a colorblind friendly palette set in addition to a set of
palettes based on the brand identities of all 14 Big Ten schools (as of
April 2024).

## Usage

``` r
guzior.pal

b10.pal
```

## Format

### `guzior.pal`

An object of class `palettes_palette` with 3 color palettes. Use
`names(guzior.pal)` to return all palette names.

### `b10.pal`

An object of class `palettes_palette` with 13 color palettes. Use
`names(b10.pal)` to return all B10 palette names.
